# JSX

- JSX  xml in js   (html是一种特定的xml)
  react 的一个亮点
  react的模版语法 类似于vue里面的template 在js里写html
  如何在jsx里面写html -> JSX = 写起来像 HTML 的 JavaScript

- npm init vite  初始化一个react项目
- vue/react 现代前端框架的特性
  响应式 reactive
  数据绑定 data binding
  组件化 component

  react 更激进，门槛高

# this 指向

## 自由变量的查找
- 编译阶段?
- 作用域链
- LexicalScope 词法作用域?
  - 词法作用域是指在词法分析阶段就确定了作用域，而不是在运行时确定作用域。
  - 词法作用域是基于函数声明的位置来确定的，而不是基于函数调用的位置。
  - 词法作用域是静态的，而不是动态的。

## 如果要拿到time.geekbang.com 怎么办？
- this 指向调用它的对象
  在面向对象的方法内**应该**可以通过this 拿到对象的属性
- js做了个不好的设计
  this 由函数调用的调用方式决定
  - 普通函数运行时，this 指向全局对象
  - 方法调用时，this 指向调用它的对象



如果一个标准函数，也就是非箭头函数，作为某个对象的方法被调用时，那么这个this指向的就是这个对象

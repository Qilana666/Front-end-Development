## 作用域
是变量查找的规则
作用域是变量的查找路径
作用域指在程序中定义变量的区域，该位置决定了变量的可见性，周期性
作用域控制着变量和函数的可见性和周期性

- 全局作用域
  在任何地方都能访问，其生命周期就是页面周期
- 函数局部作用域
  只能在函数内部访问，其生命周期就是函数执行周期
- 块级作用域
  - es5 不支持块级作用域
  - es6 支持块级作用域
